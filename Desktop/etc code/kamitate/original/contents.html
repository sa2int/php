採用情報

当社は「人から信頼されること」を信条に事業活動をしております。 スマートフォンアプリを始めとして様々な事業に取り組み、 多数の人々に信頼されるように事業を展開しております。
当社が重視しているのは「他人のことを考えられる」ことができるということです。 この考え方に共感できる方は、問い合わせフォームよりご連絡ください。

採用形式 新卒採用 / 中途採用
職務内容 スマートフォン・Webアプリケーション設計開発 / プロジェクトリーダー / プロジェクトマネージャー
必須スキル・経験 システムの設計もしくは開発実務１年以上
