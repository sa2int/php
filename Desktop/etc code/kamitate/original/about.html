株式会社ネオン・システムとは

株式会社ネオン・システムは IT業界で、人々に「信頼できる会社」 と思われる情報を提供をすることができる。 スキルを持つ社員が一丸となって、 お客様のさらなる喜びを提供する集団です。

近年はグローバルワールドネットワークとも提携し、国内のみならず、海外展開も含めて活動の場をひろげています。

History

2010年4月 株式会社ネオン・システム設立
2012年6月 時間計算アプリリリース
2014年12月 海外企業と提携してオンラインデータ共有事業をスタート
2015年1月 ソーシャルゲームアプリをリリース
